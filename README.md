# Background Change App

A simple React application built with Vite and Tailwind CSS that allows users to change backgrounds dynamically.

## Features

- Dynamic background changes
- Built with React and Vite for fast development
- Styled with Tailwind CSS
- Deployed on GitHub Pages

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ayushsahu9903/Background-change.git
   cd Background-change
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

### Deploy

The app is deployed to GitHub Pages. To deploy updates:

```bash
npm run deploy
```
